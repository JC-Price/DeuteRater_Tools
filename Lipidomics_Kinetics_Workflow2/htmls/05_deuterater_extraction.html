DeuteRater — Extraction

Purpose of this step

DeuteRater performs the core analytical task of kinetic mass spectrometry: empirically measuring deuterium incorporation into each lipid’s isotopologue envelope. This step establishes the data used to calculate the incorporable deuterium count (nL), the fraction of endogenously synthesized molecules (Asyn), and the turnover rate constant (k).

Extraction can be run independently when processing positive and negative polarities before merging them (recommended). Doing so maximizes isotopic coverage and ensures that standardized polarity identifiers from the previous step are recognized.

How to run extraction

  1. Click Open DeuteRater below.
  2. Select Lipidomics mode.
  3. Deselect all processing steps except Extraction.
  4. Modify the settings as needed (we recommend loading up "Recommended_DeuteRater_settings.yaml" which reflect the settings used in Nielsen et al., 2026).
  5. Run Extraction separately for positive and negative modes, placing the resulting .tsv files into the same directory for combined polarity analysis.

Required settings

  • Asymptote Calculation: set to Variable.
  • Calculate Experimental N-Values: On.

*The incorporable deuterium count (nL) represents the number of hydrogen positions that can exchange with deuterium under metabolic labeling. Unlike peptides, lipid nL values are not predicted from structure alone and must be determined empirically under the same biological conditions (see Nielsen et al., 2026).

Recommended Settings

Example settings file

This package includes a settings file matching the configuration used in Nielsen et al., 2026. Import Recommended_DeuteRater_settings.yaml into DeuteRater to replicate our analysis.

Output

DeuteRater produces extraction outputs per run, containing isotopologue intensities and fitted parameters required for kinetic modeling. When performing the combined-polarity workflow, ensure both polarities share the same standardized lipid identifiers from the previous step.
